# Methods in Java

    Methods : a block of reusable code that is excuted when called ()

```java
public class Main {
    public static void main(String[] args) {
        happyBirthday("mustapha", 25);
    }
    static void happyBirthday(String name, int age) {
        System.out.println("Happy Birth day " + name);
        System.out.println(name + " your age know is " + age);
    }
}
```
